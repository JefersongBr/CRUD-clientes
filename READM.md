# CRUD clientes

## Descrição

Este projeto consiste em um sistema web para registro e gerenciamento de clientes, desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento de software.

A aplicação implementa operações CRUD (Create, Read, Update e Delete) em uma única página, facilitando a compreensão do fluxo de dados entre frontend e backend.

Durante o desenvolvimento, foram trabalhadas habilidades importantes como:

Estruturação e organização de diretórios

Controle de versão com Git e GitHub

Pesquisa e utilização de documentações oficiais

Boas práticas iniciais de desenvolvimento

Este projeto faz parte da minha jornada de aprendizado na área de tecnologia e representa a aplicação prática dos conhecimentos adquiridos até o momento.

## Funcionalidades
- Cadastro de clientes
- Listar clientes
- Atualizar clientes
- Deletar clientes
